# UPsi Jam 4.0

Marathon de création de jeux vidéo de 48h non-stop au campus Arc 2 de la HE-Arc à Neuchâtel.

La Game Jam s'est déroulé du 22 au 24 mars 2024.

<a href="https://upsijam.ch/">https://upsijam.ch/</a>
